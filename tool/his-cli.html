# his-cli 脚手架

大his脚手架命令，提供模板下载、子应用模板下载、项目更新等相关功能

# 开始使用

# Install
# 安装
yarn global add his6.0-cli
# 或者
npm install his6.0-cli -g

# Create main app
# 创建新项目
his create myapp

# Start app
# 开始使用
cd myapp
yarn run serve

# 常用命令

拥有两个常用命令: create, add.

# create

创建新项目到指定目录中

his create [options]

#示例
his create myapp
his new myapp --demo
his new myapp --no-install

# options

  • --demo -- 快速生成一个简易的示例原型
  • --no-install -- 创建完成后不自动运行npm install

# add

在父项目中创建子项目

his add [subAppName]

# 示例
his add mySubApp

# update

更新子应用中主应用版本及子应用vue相关配置

his update
