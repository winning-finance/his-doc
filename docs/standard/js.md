## js 书写规范

1. 函数名、变量名使用驼峰式
2. 常量使用 CONSTANT_NAME 格式
3. vue 文件，原则上单个 vue 文件不要超过 1000行，按业务功能拆封，遵循可复用、可组合的原则。
4. 不要有 eslint 报错，有报错一定要解决，新的项目请按默认eslint 配置 规范书写代码
